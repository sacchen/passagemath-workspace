# passagemath-workspace

personal research workspace for [passagemath](https://github.com/passagemath/passagemath) contributions. specs, deep dives, and contributor resources.

---

## `/projects` — shovel-ready specs

scoped issues with root cause identified, exact files, and a verification method. written so someone can pick one up and contribute without needing to triage first.

- [oer-sage-discrete-math-passagemath.md](projects/oer-sage-discrete-math-passagemath.md) — adapt a CC-BY discrete math OER to use passagemath instead of CoCalc/SageMath
- [wasm-package-recipe.md](projects/wasm-package-recipe.md) — cross-compile a C math library to wasm32 for the browser demo
- [plot-display-plain-python-kernel.md](projects/plot-display-plain-python-kernel.md) — fix plots silently doing nothing in plain Python kernels
- [weighted-adjacency-matrix-vertices-2108.md](projects/weighted-adjacency-matrix-vertices-2108.md) — fix crash on sparse weighted graphs

## `/nerdsnipe` — deep research

- [cysignals-wasm.md](nerdsnipe/cysignals-wasm.md) — preemptible computation without OS signals
- [webgpu-math-viz.md](nerdsnipe/webgpu-math-viz.md) — GPU-accelerated math visualization in the browser
- [wasm-overview.md](nerdsnipe/wasm-overview.md) — the full picture of the passagemath WASM effort
